Consensus.OGP module

Open Geography Portal

This module provides a class OpenGeography() that connects to the Open Geography Portal API and a subclass OpenGeographyLookup() that builds and updates a JSON lookup table for the portal’s FeatureServers.

OpenGeographyLookup() is necessary if you want to make use of the SmartLinker() class from the GeocodeMerger module.

Classes:

OpenGeography: A class to connect to the Open Geography Portal API. OpenGeographyLookup: A class to build and update a JSON lookup table for the Open Geography Portal’s FeatureServers.
